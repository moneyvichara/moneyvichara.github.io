Connect to GitHub

One-time setup. Paste your GitHub token and your changes will publish directly — no downloading, no uploading.

Your token is saved only in this browser. It never appears in the code.
Don't have a token yet? See how to create one ▾
How to create your GitHub Classic Token: 1. Go to github.com/settings/tokens/new
   (If asked to choose a type, pick classic token — not fine-grained)
2. In the Note field type: Money Vichara Admin
3. Set Expiration to: No expiration
4. Under Select scopes, tick only repo (the first big checkbox)
5. Scroll down → click Generate token
6. Copy the token shown (starts with ghp_) and paste it above
⚠ Copy immediately — GitHub shows it only once. Save it in your phone notes too.
